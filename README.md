## Pytorch Bert_CasRel_RE 实体关系抽取项目

关系抽取（Relation Extraction）就是从一段文本中抽取出（主体，关系，客体）三元组，是 NLP 领域重难点任务之一，本课程选用的 CasRel 模型，是一个知名度较高的 Joint Model 关系抽取模型。实体关系抽取任务，不仅能锻炼编码能力，还能在面试中作为难点突破的创新方案，与面试官进行深入探讨。
